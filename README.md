# 🎵 Spotify Display (ESP32 + TFT)

A small project I made that shows my current Spotify track on a TFT screen using an ESP32.

---

## 🚀 Features

- Shows song name + artist  
- Connects to Spotify over WiFi  
- 3 buttons: play/pause, next, previous  
- Runs on ESP32 C3 Mini  

---

## 🧠 What I Did

- Set up Spotify API on the ESP32  
- Built a simple UI for the display  
- Wired buttons for playback control  
- Made a compact enclosure  

---

## 🧰 Parts

- ESP32 C3 Mini  
- 1.8" TFT (ST7735)  
- 3 buttons  
- Wires / soldering  

---

## 🔌 Wiring

| Part | Pin |
|------|-----|
| TFT CS   | 5 |
| TFT DC   | 2 |
| TFT RST  | 4 |
| TFT MOSI | 6 |
| TFT SCLK | 7 |
| Prev Btn | 8 |
| Play Btn | 9 |
| Next Btn | 10 |

---

## ▶️ How It Works

ESP32 connects to WiFi → gets current track from Spotify → displays it → buttons control playback.
